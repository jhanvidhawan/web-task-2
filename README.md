# To-do List
A responsive To-Do List built with HTML, CSS, and JavaScript.  
It allows users to efficiently manage their daily tasks with features like sorting, editing, marking complete, and real-time searching â€” all saved locally in the browser. (Open with live server).

# Features
- Add tasks with due date and category  
- Mark tasks as complete or incomplete  
- Edit or delete tasks  
- Search tasks in real time  
- Sort tasks by due date  
- Tasks are stored in localStorage  
- Mobile responsive design  
- Real-time clock display  

# Technologies Used
- HTML
- CSS
- JavaScript
- Font Awesome (Icons)
- LocalStorage API
